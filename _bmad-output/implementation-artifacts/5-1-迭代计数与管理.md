# Story 5.1: 迭代计数与管理

Status: review

## Story

As a 系统架构师,
I want 系统能正确管理迭代次数,
so that 可以限制无限迭代.

## Acceptance Criteria

1. **AC1: 迭代初始化**
   - Given 任务开始执行
   - When 初始化迭代状态
   - Then 迭代次数设为 0
   - And 最大迭代次数从配置读取 (默认 5)

2. **AC2: 迭代增加**
   - Given 迭代发生
   - When Generator 开始修复代码
   - Then 迭代次数 +1
   - And 记录迭代历史
