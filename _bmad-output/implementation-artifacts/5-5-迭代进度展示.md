# Story 5.5: 迭代进度展示

Status: review

## Story

As a 开发者,
I want 在 Web 页面查看迭代进度,
so that 我可以了解当前迭代状态.

## Acceptance Criteria

1. **AC1: 迭代进度显示**
   - Given 任务正在迭代
   - When 用户查看任务详情
   - Then 显示当前迭代次数 (如 2/5)
   - And 显示历次迭代的失败原因
   - And 显示当前迭代进度

### 实现说明

**已实现部分** (`internal/harness/iteration.go`):
- `GetStatus(taskID)` 返回 IterationStatus
- `IterationStatus` 包含 Current, Max, Remaining, History
- `NeedsAttention` 标记是否需要人工处理

### IterationStatus 结构

```go
type IterationStatus struct {
    TaskID          string
    Current         int              // 当前迭代
    Max             int              // 最大迭代
    Remaining       int              // 剩余迭代
    History         []IterationRecord
    NeedsAttention  bool             // 是否需要人工处理
}
```

### Web 展示

前端可以调用 GetStatus 获取迭代状态，展示:
- 迭代进度条: "2/5"
- 历史失败原因列表
- 是否需要人工处理
