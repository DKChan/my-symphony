# Story 4.4: 代码审计

Status: review

## Story

As a 系统用户,
I want 系统调用 BMAD Code Review Agent 进行代码审计,
so that 可以获得代码质量反馈.

## Acceptance Criteria

1. **AC1: 代码审计执行**
   - Given 代码实现完成
   - When Evaluator E3 开始执行
   - Then 系统调用 BMAD Code Review Agent
   - And Agent 审查代码质量
   - And 返回审计报告

2. **AC2: 问题记录**
   - Given 审计发现问题
   - When Agent 返回报告
   - Then 记录问题列表
   - And 作为失败报告的一部分
