# Story 6.4: 任务状态持久化

Status: review

## Story

As a 系统架构师,
I want 任务状态持久化到 Beads,
so that 服务崩溃后可以恢复.

## Acceptance Criteria

1. **AC1: 状态持久化**
   - Given 子任务状态变化
   - When 状态更新
   - Then 调用 Beads API 更新任务状态
   - And 保存迭代次数到 Custom 字段
   - And 保存当前阶段到 Custom 字段

2. **AC2: 状态恢复**
   - Given 服务重启
   - When 恢复进行中任务
   - Then 从 Beads 读取任务状态
   - And 恢复迭代次数
   - And 继续执行

**FRs covered:** NFR4, FR62
