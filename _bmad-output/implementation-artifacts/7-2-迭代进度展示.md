# Story 7.2: 迭代进度展示

Status: review

## Story

As a 开发者,
I want 在 Web 页面看到迭代进度,
so that 我可以了解当前迭代状态.

## Acceptance Criteria

1. **AC1: 迭代进度显示**
   - Given 任务正在迭代
   - When 用户查看任务详情
   - Then 显示迭代进度条 (如 2/5)
   - And 显示历次迭代的失败摘要
   - And 显示当前迭代状态

### 迭代状态数据

```go
type IterationStatus struct {
    Current        int               // 当前迭代 2
    Max            int               // 最大迭代 5
    Remaining      int               // 剩余迭代 3
    History        []IterationRecord // 历史记录
    NeedsAttention bool              // 需要人工处理
}
```

### Web 展示 API

前端可通过 Orchestrator.GetStatus(taskID) 获取迭代状态，展示:
- 进度条: "2/5"
- 历史失败摘要
- 是否需要人工处理
